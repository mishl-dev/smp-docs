# INFINITY SMP - Plugin Documentation

Plugin documentation for INFINITY SMP, built with [Mintlify](https://mintlify.com).

**Server:** mc.mishl.dev  
**Version:** 1.21.1 (Java Edition)  
**Type:** Survival

## Plugins Documented

### Core Plugins
- **HuskHomes** - Home and warping system

### Bukkit Plugins
- **AxShulkers** - Enhanced shulker box functionality
- **CoreProtect** - Block logging and rollback protection
- **InstantRestock** - Automatic shop restocking
- **LagFixer** - Server performance optimization
- **NoChatReports** - Chat privacy protection
- **OnePlayerSleep** - Single player sleep
- **ProtocolLib** - Plugin library
- **SkinsRestorer** - Custom skin support

## Development

Install dependencies:
```bash
bun install
```

Run the development server:
```bash
bun run dev
```
