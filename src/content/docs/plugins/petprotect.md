Protect tamed mobs from PvP.

## Features

- Prevents tamed pets from taking damage from other players
- Transferring ownership of tamed pets to another player

| Command | Description |
|---------|-------------|
| `/reassign set <name>` | Reassign a pet to the player with the given name |
